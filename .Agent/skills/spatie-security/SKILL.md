# Name: Spatie Security Expert
**Description:** Provides the Agent with specialized expertise in Role-Based Access Control (RBAC) using the Spatie Laravel Permission package.

## Core Capabilities:
* **Policy Creation:** Automatically generates Model Policies.
* **Middleware Setup:** Injects security checks into Web and API routes.
* **Role Syncing:** Ensures the database roles match the current code configuration.

## Trigger:
Activated during `/build-feature` to ensure every new module is secure by default.
